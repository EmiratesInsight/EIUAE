## 👋 About

**Emirates Insight** is a modern, independent business and insights publication covering the GCC — UAE, Saudi Arabia, Qatar, Kuwait, Bahrain and Oman.

We write for founders, operators, investors and policy-watchers who need sharp, fast, *useful* analysis of what's actually moving in the Gulf — without the press-release noise.

## 🛠️ How we're built

Emirates Insight is a fully open, edge-native publishing stack — a small proof that quality publishing doesn't need a WordPress monolith.

- **Framework:** [Astro](https://astro.build) (hybrid SSR) on **Cloudflare Pages**
- **API:** [Hono](https://hono.dev) on **Cloudflare Workers**
- **Data:** Cloudflare **D1** (SQLite at the edge) + **R2** (media) + **KV** (cache)
- **Editor:** Custom admin built on [TipTap](https://tiptap.dev) with role-based permissions
- **Newsroom AI:** Multi-provider fallback chain (OpenRouter → DeepSeek → Cerebras → Gemini → Workers AI) for assist-tooling under a strict editorial rulebook
- **SEO / discovery:** NewsArticle + Organization + BreadcrumbList schema · Google News sitemap · IndexNow auto-ping on publish · JSON Feed 1.1 · RSS 2.0
